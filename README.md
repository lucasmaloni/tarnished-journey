# Tarnished Journey

Projeto para a disciplina de Algoritmo e Estrutura de Dados que explora as possibilidades e funcionamento dos grafos. O projeto pega uma certa quantidade de dados em dados_elden_ring.csv
e monta um grafo a partir dessas informações.

Sobre os dados, basciamente é uma relação de local de graça X e tempo de deslocamento até o local de Graça Y. Cada local de graça é um Nó do Grafo e a distância é o peso da aresta entre esses nós.

O objetivo é aplicar os conhecimentos sobre Grafos e Algoritmos relacionados a essa estrutura de dados em um tópico de meu interesse.
